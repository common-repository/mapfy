WordPress Google Maps Plugin

== Description ==

WordPress Google Maps Plugin for showing Google Maps on website.



== Source Code ==

Mapfy is built with [LeafletJS](https://leafletjs.com/), an open-source JavaScript library for mobile-friendly interactive maps.

2. Or, UNZIP "mapfy.zip" folder and Upload mapfy on "wp-content/plugins/" folder via FTP or Copy and
Paste on your Local Computer
